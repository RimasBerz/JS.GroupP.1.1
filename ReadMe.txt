Браузерный кликер по тематике Minecraft

Сохраниния присутсвуют,работают через JSON,по нажатию на кнопку - сохраняеться прогресс игры(данные об игроке)
Механики,используемые в игре - ломание блоков,торговля(продажа блоков и улучшение инструментов) и 
так далее.

"1", "2", "3", "4" - для переключения инструментов
"Esc" - пауза/un-пауза
